# **App Name**: LifeLink

## Core Features:

- Smart Match-Maker Tool: Uses a tool powered by generative AI to analyze blood request urgency and automatically match hospitals with compatible donors based on location and real-time inventory levels.
- Live Blood Inventory HUD: A high-fidelity dashboard to track stock across 8 blood groups (A±, B±, AB±, O±) with visual low-stock and expiration alerts.
- Unified Access Gateway: Secure, role-based entry for Admins, Donors, and Hospitals with tailored dashboard perspectives and personalized health profiles.
- Emergency Broadcast System: Trigger instant priority notifications for hospitals needing immediate units, automatically identifying and alerting nearest matching donors.
- Donor Impact Journey: A gamified interface for donors to track their history, unlock badges for life-saving milestones, and view their next eligibility window.
- Geospatial Search Engine: Search and filter donors or banks by district and blood type using an interactive map interface.
- Demand Forecasting Engine: Visual data charts that predict upcoming blood shortages using historical donation patterns.

## Style Guidelines:

- Primary color: Vitality Red (#E61937), a sophisticated and urgent shade that reflects life and healthcare. Based on the concept of clinical professionalism and empathy.
- Background: Clinical Shell (#FFF7F8), a soft, ultra-light tinted white to provide a clean, medical-grade backdrop without harsh glare.
- Accent: Deep Mulberry (#A3196E), an analogous hue providing elegant contrast for primary actions and data highlights.
- Font pairing: 'Playfair Display' (serif) for sophisticated, high-contrast headings that command authority; paired with 'Inter' (sans-serif) for highly legible, objective body text.
- Minimalist medical pictograms and blood drop abstractions used with thin, consistent line weights.
- A spacious, editorial-style layout using deep white space and cards with subtle shadows to create depth and hierarchy.
- Smooth, choreographed micro-interactions and soft 'fade-in-up' transitions for dashboard stats and request alerts.
